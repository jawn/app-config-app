Installation
------------

Install Ruby Gems if you have not already.

The app requires at least Ruby 1.9.3 to run. 

```
$ bundle install
$ rvm install 1.9.3 # if needed
$ rvm 1.9.3
```

Running it
----------

```
$ rackup
```

Then crank up your browser, and go to [http://localhost:9292](http://localhost:9292)
